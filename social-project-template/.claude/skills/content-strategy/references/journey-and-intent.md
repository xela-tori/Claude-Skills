# Journey Frameworks & Search-Intent Modifiers

Supporting reference for Phase 3 (pillar synthesis) in the content-strategy skill.

---

## Journey stage vocabulary — adapt to the user's industry

Don't default to TOFU/MOFU/BOFU — that's B2B-SaaS-coded and doesn't fit every business. Pick the vocabulary that matches how the user's audience actually moves through the relationship:

| Business type | Journey stages |
|---|---|
| **B2B / considered purchase / SaaS** | Awareness → Consideration → Decision → Retention |
| **B2C / e-commerce** | Discover → Consider → Buy → Advocate |
| **Creator / personal brand** | Discover → Follow → Trust → Convert |
| **Community-led brand** | Attract → Engage → Belong → Amplify |
| **Local / service business** | Awareness → Trust → Book → Refer |
| **Education / course creator** | Curious → Learning → Committed → Alumni |

If none fit cleanly, invent a 3–4 stage journey that matches how this specific audience actually behaves, and name it plainly. State the chosen framework at the top of the output so downstream skills know which vocabulary to use.

## Search-intent keyword modifiers

These classify **SEO query intent** and are independent of the Journey Stage framework above. Intent is about how people phrase queries at different moments of the buying cycle — it applies whether the business is B2B SaaS, a solo creator, a local bakery, or an e-commerce brand.

The modifiers below are **common English-language starter examples, not the final list.** Actual high-value keywords vary vastly by industry, language, audience, and market — a cookware brand's "consideration" keywords look nothing like a law firm's. **Populate each stage with real keywords discovered in Phase 1** (autocomplete, PAA, GSC, keyword tools, community mining) **and Phase 2** (competitor content audit), grouped by intent. The starter examples are only there so the model knows what kind of keyword belongs in each bucket.

- **Awareness intent** — people learning a topic, no product in mind yet.
  *Starters:* "what is", "how to", "why", "guide to", "meaning of", "types of", "examples of", "beginner's guide", "introduction to", "difference between X and Y", "is [X] good for [use case]".
- **Consideration intent** — people evaluating options and comparing alternatives.
  *Starters:* "best", "top", "vs", "alternatives to", "comparison", "pros and cons", "reviews", "which [X] is best", "is [X] worth it", "[X] for [persona/use case]", "[X] or [Y]".
- **Decision intent** — people ready to act, convert, or buy.
  *Starters:* "pricing", "cost", "buy", "discount", "coupon", "near me", "demo", "free trial", "template", "case study", "[brand] reviews", "how much does [X] cost", "where to buy [X]".
- **Implementation intent** — people already using the thing and looking for help.
  *Starters:* "tutorial", "setup", "how to use", "examples", "troubleshooting", "tips", "advanced", "[X] not working", "fix [X]", "[X] step by step", "[X] best practices".

**Fill each stage with the real keywords found in research**, not these starters. The starters are training wheels — remove them in the final output and replace with discovered keywords grouped by intent.

## Content-mix % guidance by primary goal

Content mix % examples — adapt to whichever goal the user actually chose:

- **Brand awareness** → skew educational + thought leadership (top-of-journey heavy).
- **Lead generation** → balance top/middle-of-journey with conversion content.
- **Engagement / community** → skew personal, behind-the-scenes, conversation starters.
- **Traffic / SEO** → skew searchable, high-volume evergreen content across the buyer journey.
- **Conversion / sales** → skew case studies, product demos, comparison content, social proof.
- **Thought leadership** → skew original research, contrarian takes, deep analysis, proprietary data.
- **Education / course** → skew tutorials, frameworks, step-by-step guides, worked examples.

Keep pure promotional content ≤ 5–10% regardless of goal.
